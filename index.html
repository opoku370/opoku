Bronze — source of truth

Raw ingested CRM and ERP datasets land here with no business transformations. Preserves lineage before cleansing.

  • Ingest from multi-source files
  • Store as Delta tables
  • Orchestrated via notebook run

Bronze → Silver → Gold

Silver — cleaned & integrated

Standardization, missing-value handling, and CRM/ERP integration so entities can join cleanly downstream.

  • Cleansing and type consistency
  • Cross-source integration
  • Reusable transformation notebooks

Bronze → Silver → Gold

Gold — analytics model

Business-level star schema: dimension tables for customers and products, plus fact_sales with measures and keys.

  • dim_customers · dim_products
  • fact_sales — sales_amount, quantity, price
  • Surrogate keys via window functions
